비교 연산자

JS에서 부등 표시 -> !=, !== / 등호 표시는 == / =가 하나면 할당.
문자열 비교 시에는 사전 순(유니코드 순)으로 뒷 순서 문자가 더 크다고 판단한다.
차례대로 글자 순서를 비교하고 같을 시 문자가 긴 문자열이 크다고 판단.
대문자, 소문자 중에서는 소문자가 더 크다고 판단.
== -> 동등 연산자 / === -> 일치 연산자(형 변환 없이 값을 비교, 엄격한 동등 연산자)
null과 undefined는 서로 동등 비교(==)할 때만 true로 반환된다.
ex) null == 0 false / null > 0 false / null >= 0 true
비교 연산자는 Boolean값(true, false)으로 변환한다. 비어있지 않은 문자열 "0"은 모두 true로 반환된다. 비어있는 문자열 "" 띄어쓰기 없이.
