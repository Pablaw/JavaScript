기본 연산자와 수학

단항, 이항, 피연산자 나머지 연산자 % / 거듭제곱 연산자 ** + 연산자의 기능

JS에서 이항 연산자 +는 문자열을 연결하는 기능을 한다.
그리고 피연산자 중 하나라도 문자열이 있으면 나머지도 문자열로 변환된다.
연산은 왼쪽에서 오른쪽으로 순차적으로 진행된다. 문자열이 같이 있는 경우도 해당.
ex) alert( 2 + 2 + '1' );의 경우 '221'이 아니라 '41'이 출력된다.
단항 + 연산자는 Number(value)와 동일한 기능을 할 수 있다.
ex) let apples = '2'; let oranges = '3';
alert( apples + oranges ); // 23
alert( +apples + +oranges) // 5

연산자 우선순위 - 동일한 단항 연산자는 이항 연산자보다 우선 순위가 높다. 증감, 감소 연산자는 다른 연산자 보다 우선 순위가 높기 때문에 평가가 먼저 이루어진다. '코드 한 줄엔, 특정 동작 하나'에 관련된 내용만 작성하는 게 좋다.
